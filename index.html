Select Trigger

The trigger is the event that causes your bot flow to run.

Discord Interactions

Slash Command

Triggered by a slash command

  • /hello - Greets the user
  • /ban - Ban a member
  • /kick - Kick a member
  • /info - Show server info
  • /help - Show help menu

Context Menu

Triggered by a user/message context menu option

  • View Profile
  • Report Message
  • Mute User
  • Message Logs
  • Quick Ban

Button Click

Triggered by a button click below a message

  • Confirm Action
  • Cancel
  • Open Modal
  • React with Emoji
  • Join Giveaway

Modal Submit

Triggered by a form/modal submission

  • Feedback Form
  • Bug Report
  • Request Support
  • Apply for Role
  • Submit Suggestion

Dropdown Select

Triggered by a dropdown selection

  • Choose Category
  • Select Role
  • Set Status
  • Filter Results
  • Pick Language
Discord Messages

Message Sent

Triggered by a sent message

  • Auto-reply
  • Log message
  • Check for spam
  • Translate text
  • Analyze sentiment

Message Deleted

Triggered by a deleted message

  • Log deletion
  • Notify moderators
  • Recover message
  • Tag user
  • Store backup

Poll Vote Added

Triggered by a user voting in a poll

  • Thank voter
  • Update live results
  • Give role
  • Send DM
  • Log vote
